# Lander
Python PyGame lunar lander game


# Control
> # Maneuver
> **A** - rotate left  
> **D** - rotate right  
> **W** - rotate accelerate  
> **SPACE** - stage cut off  
> # RCS
> **KEYPAD 0** - release mode  
> **KEYPAD 1** - stabilization  
> **KEYPAD 2** - prograde stabilization  
> **KEYPAD 3** - retrograde stabilization  
> # Other
> **Enter** - spacewalk (only when it landed)
